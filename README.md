# Epic-space-battle-music
Simple mod that replaces the main two, lackluster and repetitve space combat themes with "Hero theme" and "Spore war" from spore hero's ost to make interstellar combat a bit more "epic". Will work with stream safe music (and should be copyright free) and works wonderfully with share safe music and replaces the space combat tracks in galactic adventures when used with the mod.
